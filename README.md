AntiAFK Utility

Утилита для симуляции активности в играх и приложениях, чтобы предотвратить автоматический кик по бездействию (AFK).
Основные возможности:

    Гибкая настройка: Указывай любые клавиши для нажатия прямо в веб-интерфейсе.

    Умный рандом: Использование случайных интервалов и комбинаций клавиш для обхода базовых проверок античитов.

    Веб-панель: Удобное управление запуском, настройка таймингов и переключение тем (светлая/тёмная) в браузере.

    Фоновый режим: Программа работает в системном трее, не мешая основному процессу.

    Быстрый хоткей: Зажми клавишу 'L' (Д) на 5 секунд прямо в игре, чтобы быстро запустить или остановить работу макроса.

Как использовать:

    Скачай актуальную версию app.exe из раздела Releases.

    Запусти файл. Браузер автоматически откроет панель управления.

    Настрой нужные клавиши (через запятую) и временные интервалы.

    Нажми "ЗАПУСТИТЬ ЗАЩИТУ".

    Программу можно свернуть — она останется в трее.

Техническая информация:

    Написано на Python с использованием Windows API.

    Не требует установки Python или дополнительных зависимостей.

    Автоматически добавляется в автозагрузку Windows для удобства.
